Encryption Algorithm for AES

Implemented encryption algorithm of the Advanced Encryption System (AES) for 128 bit key, assuming the plaintext input message is always the length of 128 bits.

The goal was to Implement encryption algorithm with plaintext input message of 128 bits with two test cases. The implementation contains nine main rounds with one final round. In each main round, it performs four operations while the final round only performs three. The operations are stated in order byte substitution, shift rows, mix columns, and add round key. The final round skips the mix columns operation.

  • The first step to my implementation is reading both plaintext and key files into an array. The plaintext array is read into a multidimensional array where x (left and right) and y (up and down) are used to make a 4x4 grid. The values are in increasing order from column to column. Before the main rounds begin, it performs add round key.
  • Byte Subsitution is implemented to utilize S-box to replace every byte in the message. I implemented it in a way that replaces the byte by its value and not indexed by row and column.
  • Shift Rows is implemented to shift rows of the matrix left, increasing as it goes down the rows. It uses a temporary array to hold the shifted values which replaces the values in matrix.
  • Mixed Columns is implemented to replace matrix values by using matrix multiplication. The matrix is multiplied by a constant matrix into a temporary matrix. The values of the temporary matrix replaces the original matrix.
  • Add Round Key is implemented to perform exclusive or operations between the values in the matrix and key array.
  • The last step to my implementation is writing the values in the matrix a file. This is done by reading the values in the matrix into an array which is written into a file.

The same key is used everytime it performs the exclusive or operation each round.
